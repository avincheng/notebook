# HTML 常用 DTD 声明

## HTML 5

```html
<!DOCTYPE html>
```

## HTML 4.01

HTML 4.01 Frameset，该 DTD 包含所有 HTML 元素和属性，包括展示性的和弃用的元素（比如 font），且允许框架集（Framesets）。

```html
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">
```
